---
title: "nexus"
description: "Launch the Nexus interactive terminal dashboard."
---

`pylon nexus` launches the full-screen interactive terminal dashboard for managing pylons, controlling the daemon, and monitoring jobs.

## Usage

```bash
pylon nexus
```

For complete documentation on the Nexus interface, keybindings, and features, see [Nexus](/concepts/nexus).

<Note>
Nexus requires a terminal with at least 60 columns. If the window is too narrow, it displays a resize prompt.
</Note>
